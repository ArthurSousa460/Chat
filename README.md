<h1>Chat</h1>

Código feito na disciplina de python no curso de redes de computadores no IFMA (Intituto de Ciência de Tecnológia do Maranhão)

<h2>Tecnologias usadas</h2>
<ul>
  <li>Python</li>
  <li>Socket</li>
  <li>Threading</li>
</ul>
